    T11 CF1110F Nearest Leaf

    题意翻译

    给定一棵以1 为根的n个节点有根树

    1不是叶子

    给定m次询问

    形如 v l r

    输出以v为起点 终点编号为l - r以内的叶子中 v到叶子最近的距离

    注意：如果我们 DFS 这棵树，对于每个点都递增枚举儿子节点，每访问到一个节点就记录其编号，那么得到的序列刚好为 1 到 n。

    题目描述

    Let’s define the Eulerian traversal of a tree (a connected undirected graph without cycles) as follows: consider a depth-first search algorithm which traverses vertices of the tree and enumerates them in the order of visiting (only the first visit of each vertex counts). This function starts from the vertex number 1 1 1 and then recursively runs from all vertices which are connected with an edge with the current vertex and are not yet visited in increasing numbers order. Formally, you can describe this function using the following pseudocode:

    You are given a weighted tree, the vertices of which were enumerated with integers from 1 1 1 to n n n using the algorithm described above.

    A leaf is a vertex of the tree which is connected with only one other vertex. In the tree given to you, the vertex 1 1 1 is not a leaf. The distance between two vertices in the tree is the sum of weights of the edges on the simple path between them.

    You have to answer q q q queries of the following type: given integers v v v , l l l and r r r , find the shortest distance from vertex v v v to one of the leaves with indices from l l l to r r r inclusive.

    输入输出格式

    输入格式：

    The (i−1) (i - 1) (i−1) -th of the following n−1 n - 1 n−1 lines contains two integers pi p_i pi and wi w_i wi ( 1≤pi<i,1≤wi≤109 1 \leq p_i < i, 1 \leq w_i \leq 10^9 1≤pi<i,1≤wi≤109 ), denoting an edge between vertices pi p_i pi and i i i with the weight wi w_i wi .

    It’s guaranteed that the given edges form a tree and the vertices are enumerated in the Eulerian traversal order and that the vertex with index 1 1 1 is not a leaf.

    The next q q q lines describe the queries. Each of them contains three integers vi v_i vi , li l_i li , ri r_i ri ( 1≤vi≤n,1≤li≤ri≤n 1 \leq v_i \leq n, 1 \leq l_i \leq r_i \leq n 1≤vi≤n,1≤li≤ri≤n ), describing the parameters of the query. It is guaranteed that there is at least one leaf with index x x x such that li≤x≤ri l_i \leq x \leq r_i li≤x≤ri .

    思路：

    既然没说强制在线，那么我们考虑离线

    我们把询问挂到对应的点上

    考虑在 $dfs$ 的过程中，维护每个点到 $u$ 的距离

    显然是在进入点 $v$ 的时候，$v$ 及其子树的距离减 $w$，外面的点的距离加 $w$，这个东西显然可以直接用线段树

    T12 洛谷 P4747 [CERC2017]Intrinsic Interval

    题意翻译

    题目描述

    对于正整数 1,2,3⋯n1,2,3 \cdots n1,2,3⋯n 的一个排列 π\piπ，若它的一个子串 π[a..b]\pi[a..b]π[a..b] 排序后是连续正整数，则称 π[a..b]\pi[a..b]π[a..b] 是一个“区间”。例如对排列 pi=3,1,7,5,6,4,2pi={3,1,7,5,6,4,2}pi=3,1,7,5,6,4,2，子串 π[3..6]\pi[3..6]π[3..6] 是一个“区间”（因为它包含 4,5,6,74,5,6,74,5,6,7），π[1..3]\pi[1..3]π[1..3] 则不是。

    一个子串的“本征区间”是包含该子串的最短区间。“包含”是指：若 π[x..y]\pi[x..y]π[x..y] 的本征区间是 π[a..b]\pi[a..b]π[a..b]，则 a≤x≤y≤ba \le x \le y \le ba≤x≤y≤b。

    题目描述

    Given a permutation π\piπ of integers 111 through nnn, an interval in π\piπ is a consecutive subsequence consisting of consecutive numbers. More precisely, for indices aaa and bbb where 1≤a≤b≤n1 \le a \le b \le n1≤a≤b≤n, the subsequence πab=(πa,πa+1,…,πb)\pi^b_a = (\pi_a, \pi_{a+1}, . . . ,\pi_b)πab=(πa,πa+1,…,πb) is an interval if sorting it would yield a sequence of consecutive integers.

    思路：

    我们来分析一下连续段的性质

    首先我们知道两个连续段的交一定是连续段

    然后我们又知道有交的连续段的并也是连续段

    那么我们发现对于能够包含 $[x_i,y_i]$ 的长度最小的连续段一定是最大的 $l$，使得存在一个最小的 $r$，$[x_i,y_i]$ 被 $[l,r]$ 包含

    由上面的性质不可能存在一个更小的 $l’$，其符合条件的 $r’$ 小于 $r$

    那么我们现在需要知道如何判断 $[l,r]$ 是连续段

    我们考虑统计二元组 $(u,v)$ 的数量，其中 $|u-v|=1$，如果 $[l,r]$ 是连续段的充要条件是 $[l,r]$ 的二元组数量是 $r-l $

    那么我们现在考虑将所有询问离线，然后对右端点进行扫描线，用线段树维护二元组数量

    为了方便计算，左端点 $l$ 的初值为 $l$，这样如果对于当前右端点 $r$ 是连续段，则权值是 $r$

    注意到这样还需要每次对所有询问都判断一遍

    然而实际上不需要，只需要把询问按照左端点从大到小排序，然后依次询问即可
